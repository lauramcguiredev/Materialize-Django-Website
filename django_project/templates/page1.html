Materialize

About
  • Materialize is a popular front end framework using CSS and Javascript.
  • Some websites using Materialize: paycomonline.com, pricechanges.xfinity.com, and lynx-qa.wnba.com.
Strengths
  • Great for new developers looking to create a great looking website because it enforces material design concepts.
  • Good for developers looking to easily produce responsive apps.
Weaknesses
  • Uses large file sizes which can give it more overhead than desired.

Django

About
  • Django is a very widely used, python based, back-end framework.
  • Some popular websites running off Django: Disqus, Instagram, Youtube, Bitbucket, and Dropbox.
Strengths
  • Django allows management of all its application's data using only Python. This is possible because Django uses an object-relational mapper to transform database structures into Python classes.
  • The framework comes set up with many features including user authentication, content management, and much more.
  • Has great, built in security protocols so developers don't need to be concerned with common security vulnerabilities.
  • Scales easily due to its composite architecture. Definitively separated components mean hardware resources can be added as needed to any level.
  • Django makes the development process much more streamlined and fast- great for developers working with a strict deadline.
  • Great for all types of projects, from small to large, because of its scalability and robust features.
  • Great for developers who value organization because it encourages tidy and logical design.
Weaknesses
  • Uses complicated syntax for its URL routing patterns.
  • Has a monolithic structure which forces its users to follow specific rules and takes away flexibility.
  • Commitment to backwards compatibility means more overhead as time goes on.
